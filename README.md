# hormunity_test
Тестовое задание для Hormunity. Проект содержит в себя верстку сайта в MVC-приложении, заложен SOLID подход. Заказы сохраняются в MySQL, но есть подстраховка на уровне сессии. Корзина и избранное сохраняется в сессии. Можно запустить через Docker или локально(например XAMPP). В идеале товары хранить в БД, но сейчас захардкожено, с целью отображения функционала.
